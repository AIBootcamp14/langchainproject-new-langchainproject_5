# 통합 시나리오: 예상 질문 모음

## 📌 개요

이 문서는 논문 리뷰 챗봇의 7가지 도구를 활용한 다양한 사용 시나리오와 예상 질문을 정리합니다.

**도구 목록**:
- `general`: 일반 답변 (LLM)
- `search_paper`: RAG 논문 검색
- `glossary`: RAG 용어집
- `web_search`: 웹 검색
- `summarize`: 논문 요약
- `text2sql`: Text-to-SQL (데이터베이스 쿼리)
- `save_file`: 파일 저장

## 🔄 다중요청 도구 전환 패턴
