**RAG(검색-증강 생성)가 뭐냐면요** 🔍✍️  
RAG는 "중요한 정보를 찾아 답변에 추가하는 AI 기술"이에요!  
- AI가 대답할 때 기억만 믿지 않고, 외부 자료(예: 책, 논문)를 검색해 **정확한 정보**를 바탕으로 답변해요  
- "기억력 + 검색력"을 합친 똑똑한 시스템이라고 생각하면 돼요 ✨
