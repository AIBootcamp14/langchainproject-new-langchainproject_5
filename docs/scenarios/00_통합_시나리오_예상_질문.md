# 통합 시나리오: 예상 질문 모음

## 📌 개요

이 문서는 논문 리뷰 챗봇의 6가지 도구를 활용한 다양한 사용 시나리오와 예상 질문을 정리합니다.

**도구 목록**:
- `general`: 일반 답변
- `search_paper`: RAG 논문 검색
- `glossary`: RAG 용어집
- `web_search`: 웹 검색
- `summarize`: 논문 요약
- `save_file`: 파일 저장

---

## 🎯 시나리오 1: 초보자 학습 경로

**사용자 프로필**: AI/ML 입문자
**목표**: 기초 개념부터 논문까지 단계적 학습

### 예상 질문 흐름

**1단계: 기본 개념 이해** (도구: `general`)
```
"머신러닝이 뭐야?"
```

**2단계: 핵심 용어 학습** (도구: `glossary`)
```
"지도 학습이 뭐야?"
```

**3단계: 관련 논문 검색** (도구: `search_paper`)
```
"지도 학습 관련 주요 논문 알려줘"
```

**4단계: 내용 저장** (도구: `save_file`)
```
"지금까지 설명한 내용 파일로 저장해줘"
```

**예상 도구 호출 순서**:
```
general → glossary → search_paper → save_file
```

## 🎯 시나리오 2: 최신 논문 리서치
