# 시나리오: 웹 검색 (Web Search)

## 📋 도구 설명

**도구명**: `web_search`
**목적**: 외부 웹 검색을 통한 최신 논문 및 정보 수집

### 주요 기능
- Google Scholar, arXiv 등 웹 검색
- 최신 논문 정보 수집
- 실시간 정보 반영
- 검색 결과 기반 답변 생성

---

## 🎯 사용 시나리오

### 시나리오 1: 최신 논문 검색

**사용자 상황**: DB에 없는 최근 발표된 논문 정보가 필요함

**사용자 질문**:
```
"2024년에 발표된 LLM 논문 중 최신 연구는 뭐가 있어?"
```

**Agent 동작**:
1. Router가 "최신", "2024년" 키워드 감지
2. `web_search` 도구 선택
3. 웹 검색 API 호출
4. 검색 결과 파싱
5. LLM이 결과를 종합하여 답변

**예상 답변**:
```
2024년 발표된 주요 LLM 논문:

1. **GPT-5: Multimodal Foundation Models**
   - 발표: 2024년 3월
   - 주요 내용: 텍스트, 이미지, 오디오 통합 학습
   - 성능: GPT-5 대비 20% 향상

2. **LLaMA 3: Efficient Large Language Models**
   - 발표: 2024년 5월
   - 주요 내용: 오픈소스 LLM, 70B 파라미터
   - 특징: 상업적 활용 가능

3. **Claude 3 Opus**
   - 발표: 2024년 3월
   - 주요 내용: 200K 컨텍스트 윈도우
   - 벤치마크: 대부분의 작업에서 GPT-5 능가

🔗 검색 출처: Google Scholar, arXiv
⏰ 검색 시각: 2024-11-03 10:30
```

## 🔧 내부 구현

### 검색 API 호출
```python
# 웹 검색 실행
search_results = web_search_api(query)

# 결과 파싱
for result in search_results:
    title = result["title"]
    snippet = result["snippet"]
    url = result["url"]
```

### 로깅
- **tools/web_search.log**: 검색 쿼리 및 결과
- **prompts/system_prompt.txt**: 검색 결과 종합 프롬프트
- **prompts/user_prompt.txt**: 검색 결과 + 사용자 질문
